# add_comment.py

## Use

`add_comment.py` **MUST** be used when the issue contract is unclear or when follow-up discussion is
needed on the issue thread.

## Required Arguments

- `--issue`
- comment text

## Optional Arguments

- None.

## Effect

Adds a new comment to the target issue.
On success, the command prints `Action succeeded`.
